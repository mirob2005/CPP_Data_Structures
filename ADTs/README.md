# *ADTS:*
### Queue
- head/tail ptrs, remove old at head, append new to the front

### Stack
- push/pop at head

### Priority Queue
- uses an vector-based Binary Heap

## *Binary Heap:*
### *Operations:*
- traverseBFS
- insert (with heapifyUp to ensure heap property)
- delete (with heapifyDown to ensure heap property)
- peek (get max value)

### *Types:*
- Vector Structure
